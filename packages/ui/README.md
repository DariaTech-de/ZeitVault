# packages/ui

**Zweck:** Geteiltes Designsystem und UI-Komponentenbibliothek von ZeitVault – zwei Erlebnisse (Admin-Konsole und Mitarbeiter-Self-Service), ein konsistentes Designsystem. Bündelt wiederverwendbare Komponenten, Tokens und Layout-Bausteine für das Web-Frontend.

**Geplanter Tech-Stack:** React 19.2 + Tailwind CSS v4 + shadcn/ui auf TypeScript 5.x / Node.js 24 LTS, eingebunden über Turborepo 2.x + pnpm 10. Hell-/Dunkelmodus, konsistente Komponenten, ruhiges modernes Interface; primär konsumiert von `apps/web` (Next.js 16).

**Barrierefreiheit (MUSS):** Komponenten erfüllen WCAG 2.1 AA (Tastaturbedienung, Kontraste, Screenreader) – auch wegen BFSG.

**Status:** Platzhalter – Implementierung folgt in Phase 1 gemäß Paragraf 18.

**Architektur:** siehe [Paragraf 14 – UI/UX-Konzept](../../docs/ARCHITEKTUR.md#14-uiux-konzept).
